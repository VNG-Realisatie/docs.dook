# Themas Datavoorziening Ondermijning Op de Kaart (DOOK) - VNG Realisatie

De Datavoorziening Onregelmatigheden Op de Kaart (DOOK) bevat meerdere themaâ€™s:
- [Autobranche](./autobranche)
  Dit thema is ontworpen om gemeenten te helpen met het opstellen van een informatiebeeld voor de aanpak van ondermijnende criminaliteit in de autobranche.  
- [Bedrijventerreinen](./bedrijventerreinen)
  Dit thema is ontworpen om gemeenten te helpen met het opstellen van een informatiebeeld voor de aanpak van ondermijnende criminaliteit op een bedrijventerrein.

DOOK sluit met deze thema's aan op de methodiek zoals beschreven in de [Handreiking Informatiebeeld Autobranche](https://kennisplatformondermijning.nl/files/view/52f04179-a7ce-438f-99e8-ef8111896e71/handreiking-informatiebeeld-autobranche.pdf) en [Handreiking Informatiebeeld Bedrijventerreinen](https://kennisplatformondermijning.nl/files/view/5b06b150-56ff-4484-bfe1-0e26adff632f/handreiking-informatiebeeld-bedrijventerreinen.pdf) van Taskforce RIEC en Avans Hogeschool.
